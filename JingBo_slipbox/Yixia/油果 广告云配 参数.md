# 油果 广告云配
## 开屏&全局
```
//---------------------------------------------------------------------------//
//  开屏广告，全局配置
youguo.ad.open

isOpenScreen  //是否开启开屏广告
isOpenFeed    //是否开启信息流广告
adOpenScreenTime //开屏广告倒计时
adOpenScreenAgainTime// app处于活跃状态，切换前后台，再次请求开屏广告间隔时间
displayVideoTime // 全局，两次 贴片广告 之间的间隔时间
delayTime // 全局，广告 延迟 出现 倒计时
displayPerDaycount // 全局，贴片广告最多展示条数，按天
feedShowAdLoaction // 全局，信息流 间隔 几个视频 出广告
feedFirstAdPosition //全局，信息流 第一刷 第几位 出广告

//---------------------------------------------------------------------------//
```
## 前贴
```
//   前贴
youguo.ad.pre.patch  

isOpen    //是否开启
videoDuration	//视频内容长度大于X秒（可云配，默认60s）时，才请求中插广告
displayPercent //展示概率

//---------------------------------------------------------------------------//
```
## 中贴
```
//   中贴
youguo.ad.middle.patch

isOpen    //是否开启
requestStart //中贴片广告请求区间：开始，播放进度达到视频时长的百分比：默认 10
requestEnd   //中贴片广告请求区间：结束，播放进度达到视频时长的百分比：默认 40
displayStart  //中贴片广告展现区间：开始，播放进度达到视频时长的百分比：默认 50
displayEnd    //中贴片广告展现区间：结束，播放进度达到视频时长的百分比：默认 90
videoDuration		//视频内容长度大于X秒（可云配，默认60s）时，才请求中插广告
displayPercent		//生效概率

//---------------------------------------------------------------------------//
```

## 后贴
```
//   后贴
youguo.patch.ad

isOpen    //是否开启
progress  //后贴片广告 请求时机播放进度达到视频时长的百分比：默认 50
videoDuration		//视频内容长度大于X秒（可云配，默认60s）时，才请求中插广告
displayPercent		//生效概率

//---------------------------------------------------------------------------//
```
